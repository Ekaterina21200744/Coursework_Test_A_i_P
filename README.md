# Coursework_Test_A_i_P
Дипломный проект по автоматизации тестирования веб-приложения на Python с использованием Pytest

Автоматические тесты на проверку функционала  Продукт: официальный сайт оператора FUN&SUN/
Сайт для поиска и бронирования туров, отелей, авиабилетов, круизов. 
Cайт https://fstravel.com/


Для запуска тестов для формирования отчета нужно:
1. Установить необходимые пакеты в VSC, такие как:
pytest
selenium
allure-pytest
requests
2. Запустить тесты  с генерацией отчетов Allure командой "pytest .\lesson_10\form\ --alluredir=allure-results"/

Для просмотра отчета нужно запустить в терминале команду "allure serve allure-results"
В браузере откроется отчет
